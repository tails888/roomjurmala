# ROOM Jurmala

Static website for ROOM Jurmala, an event and activity space in Jurmala.

## Project Structure

- `index.html` - page markup and SEO metadata
- `styles.css` - site styling
- `script.js` - language switching and interactive behavior
- `assets/images/brand/` - logos and brand marks
- `assets/images/icons/` - favicons and app icons
- `assets/images/gallery/` - space and event photography
- `assets/images/hero/` - decorative hero cutouts
- `assets/images/reviews/` - localized review call-to-action images
- `assets/images/social/` - social sharing preview images
- `assets/videos/` - active site videos
- `assets/videos/archive/` - legacy video exports kept for reference

## Deployment

This is a plain static site. It can be served from any static host by publishing the repository root.
